小明的自我介绍

  • 021800813。

  • 小明 这个昵称吧，说起来年代好像有些久远了，依稀记得是小学时候（大概是五六年级吧）换的网名然后一用就用到了现在，也不打算改了。

  • 要说兴趣爱好也不多，能说上来的也就几个。坚持打篮球好多年了，但是好像上了大学之后打得越来越少了（老懒狗了）。平常没事喜欢到处乱逛，学校周边的大街小巷我都走过（所以封校让我很是难过）。生活中偶尔折腾折腾一些稀奇古怪的东西（很多东西被我拆了又装，装了又拆，属实闲的蛋疼）

  • 说句实话目前食堂里还没有真正喜欢吃的菜，好像每天都是凑合着填饱肚子（u1s1现在觉得家里的菜真香）。

  • 享受生活。

思考

  1. 回想一下你初入大学时对计算机专业的畅想。

    • 当初你是如何做出选择计算机专业的决定的？

    可能是受到表哥的影响，小时候经常被计算机吸引，觉得一个人只要有足够的能力，他就可以在计算机上做出任何他想做的事情（小时候真的觉得很神奇，感觉跟魔法一样）。因为人比较闷骚，所以我一直以为靠技术吃饭是最适合我的出路，只要有技术，就饿不死自己。从高中开始就盼着将来上大学学个计算机相关的专业，将来能够从事相关的工作。无奈高考分数不够，被调剂到了别的专业。在知道自己被调剂的时候就下定决心，大一无论如何都要转专业来这里，当然最终也是如愿以偿了。

    • 你认为过去两年中接触到的课程是否符合你对计算机专业的期待，为什么？

    来到这的两年，接触到了很多从未了解过的东西。说实话以前把计算机这个领域看得太小了，直到真正接触了之后才发自己是井底之蛙。现在越学就觉得自己不懂的越多，总有一大片未知的东西需要自己去探索。但总的来说这两年学习的内容也还算符合自己的预期，按部就班地也都学下来了。

    • 你觉得计算机是你喜欢的领域吗，它是你擅长的领域吗？

    对计算机的兴趣一直都挺浓厚的，相比于其他领域计算机应该是我比较擅长的（自我感觉良好），因为自己爱折腾而且逻辑思维也还行，学起来不算太费劲。

    • 将来你会选择从事计算机相关的工作吗？是的话给出你想去的城市、公司和岗位，否的话给出原因。

    将来肯定会从事相关工作的，毕竟除了本专业也没有一技之长来养活自己了。将来如果有能力的话希望去大城市闯一闯，首选应该是杭州、深圳、广州，一方面是因为这些城市让我感受到了一些独特的魅力（钱多），另一方面也是为了去学习更多的东西，去拓展自己的视野，为了更好地养活自己。

    公司的话当然是大厂啦（年轻的时候多赚点钱学点本事总是好的），年纪大些了再跳槽去别的公司或者自己创业。

    岗位的话从后端开发干起吧，一步步往上爬，希望以后能当个优秀的架构师，甚至技术总监（梦想还是要有的）。

  1. 即将大三的你，对照前人们走过的路和描述未来发展，现在的你。

    • 自我感觉你已经具备的专业知识、技能、能力有哪些？

    经过了两年时间的学习，我掌握了 c/c++、PHP、Python 的基本使用方法，知道如何用 git 来进行版本控制，了解了一些前端的基本知识。

    • 离成为一个合格的计算机专业本科毕业生，在专业知识、技能、能力上还差距哪些？

    想要成为一个合格的本科毕业生，首先要把剩余的专业课知识学好，其次在课外也要学习一些热门的技术，积攒一些项目经验，为将来打下基础。

  1. 大三是一个人生选择的十字路口，考研、工作、考公、出国，不同的选择在大三就有不同的努力方向。而无论考研还是工作的每条路径，也有许多不同的分支。

    • 对照以上你阅读的前人们的经历，你的选择是什么？

    我选择考研。一方面考研已成为一种趋势，另一方面也是想通过研究生三年的锻炼使自己的水平跟上一个层次，将来工作所受的限制也更少。

    • 在这种选择下，你认为你相比其他同学来说有何优势，有何劣势？

    我认为我对已经明确的目标有着良好的执行能力，希望在以后备战考研的时间里能够细化自己的目标，凭借自己的努力一战上岸。

    • 针对你的选择，你给自己的大三设定的规划安排是什么？

    大三上学期首要任务还是把专业课学好，利用好课余时间再学些自己感兴趣的东西（毕竟下学期也没那么多时间了），大三下就要专注于考研复习了（想想都觉得累）。

  1. 如何看待国有品牌的自主创新？如何看待自研型创新与整合优秀供应链创新这两种方式。

    国产品牌有自主创新的能力才能够突破技术壁垒，打破国外垄断。最近华为的芯片断供事件又一次给我们敲响警钟，可见核心自主知识产权的重要性。自研型创新所需要的的投入是远大于供应链创新的，一般的中小型企业很难做到。但是自研型创新对于国家来说有着十分重大的意义，尤其是这些年来国家也一直在强调的”卡脖子“的问题。自研型创新也不应当只由大型企业去带头研发，国家应当出台一些相关政策鼓励中小企业参与到研发当中去。

    当然也不是说整合供应链就意味着低端，供应链整合商也同样需要多方面的创新。小米就是近几年来最典型的例子之一，小米不仅仅在手机方面产生了许多独特的想法，近几年在智能家居方面,也给我们带来了许多惊喜。小米通过创新把互联网和传统家具整合在了一起，为我们的生活带来了极大的便利，引领了一波智能家具热潮。以我看来整合供应链的这种创新方式同样也是不可或缺的，或者说在保证自主创新的基础上，同时也要发展好整合供应链创新，以创造更丰富更优秀的产品。

未来的期望

  1. 你愿意为了一个你感兴趣的、或得到实战锻炼的项目熬夜吗？你想脱离小菜鸟的level吗？如何评价上上上、上上、上一届同学说的“真正和我的队友们一起见识了凌晨四点的福大。一起熬夜，一起吐槽，一起说说笑笑，一起打代码。一切一切的都是“美好”的回忆。

    如果是有意思或者是能够提升自己水平的项目的话那当然是值得一熬的，毕竟我还年轻。如果能够在这样轻松愉悦的氛围下，一起工作学习，相互交流，分享经验，那确实是一件十分美好的事情。

  2. 针对上述问题1、2、3，你对这门课的期待是什么？你打算平均每周拿出多少个小时用在这门课上，以达成你的期待以及你在 1 或 2 或 3 上的目标？

    我希望在这门课程里我能够把我之前的所学所想运用在实际的项目开发中，把知识转化为实战经验，期待着和队友们一起完成项目后收获满满的成就感。

    如果按照平均一天两个小时来算的话，每周应该可以花费14个小时左右用于学习这门课程。
